# 🎮 Educational Games Hub (教育性 HTML 遊戲中心)

這個儲存庫集合了 gpsftuEbsl 的多個小型 HTML 遊戲和互動專案，旨在提供一個國小高年級數學與國中程式設計的學習與娛樂平台。所有遊戲皆透過 GitHub Pages 部署和託管。

## 💻 專案架構與運行

本專案採用靜態網頁結構，便於托管和維護：

1.  **`index.html`**: 專案的入口點和主選單頁面。
2.  **`games.css`**: 主選單的樣式表。
3.  **`/games` 資料夾**: 包含所有單獨的遊戲 HTML 檔案 (`*.html`) 及其相關資源 (`.js`, `.css`, 圖片等)。

### 本地運行方法

如果您想在本地端進行開發或測試，只需執行以下步驟：

1.  克隆本儲存庫到本地電腦。
2.  使用瀏覽器直接開啟 **`index.html`** 檔案。

### GitHub Pages 部署

本專案已配置為使用 GitHub Pages 進行自動部署。請確保您的儲存庫設定：

* **Source (來源)**: `main` 或 `master` 分支。
* **資料夾**: `/ (root)`。
